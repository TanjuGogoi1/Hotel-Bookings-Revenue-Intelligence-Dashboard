# Hotel-Bookings-Revenue-Intelligence-Dashboard
This project analyzes hotel booking and revenue data from The Kaziranga Address to provide actionable insights for business decision-making.

What this project does:
--Tracks occupancy rate & revenue trends
--Analyzes guest behavior & booking patterns
--Evaluates payment methods performance

Key Metrics:
--Occupancy Rate
--ADR (Average Daily Rate)
--Total Revenue
--Booking Count
👉 Tools Used:
--Power BI
--DAX
--Data Modeling
👉 Key Insight:
--Identified high-performing booking channels like Booking.com
--Helped optimize pricing & marketing strategies
